Description: Functions for working with guilds in eScript.

CreateGuild()
Explanation
Creates a new Guild Object.
Return values
A Guild Reference to the new guild.
Related
Guild



DestroyGuild( guild )
Parameters:
NameType
guildGuild Reference
Explanation
Destroys an empty guild.
Notes: Must first empty the guild of members, allies, and enemies (see guildref methods)
Return values
1 on success
Errors
"Guild has members"
"Guild has allies"
"Guild has enemies"
"Guild has disbanded"
"Invalid parameter type"
Related
Guild



FindGuild( guildid )
Parameters:
NameType
guildidinteger
Explanation
Gets a Guild Reference for a given Guild ID number
Return values
Guild Reference on success
Errors
"Guild not found"
"Invalid parameter type"
Related
Guild



ListGuilds()
Explanation
Gets a list of all guilds.
Return values
