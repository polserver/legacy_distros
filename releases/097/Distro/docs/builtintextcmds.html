List of Built-In Text Commands. Note many of these are OLD, and/or useless. Originally meant for development team use only, some may be useful to admins out there.
  • "armor" prints your armor coverage to the screen.
  • "constat" sends connection statuses of all clients to you
  • "i_repdata" sends reputation system data for a targetted char
  • "t_ident" sends serial and graphic data for a targetted obj
  • "integ_item" sends status of world item integrity
  • "integ_chr" sends status of world character integrity
  • "i_s_item_integ" sends status of current zone's item integrity
  • "list_crit" sends list of critical scripts
  • "list_scripts" sends list of scripts ran or running
  • "log_profile" prints script profile to pol.log
  • "log_profile_clear" prints script profile to pol.log and clears counters
  • "orphans" prints orphaned object data
  • "procs" prints process data
  • "t_props" prints cprop info for targetted obj
  • "resendchars" sends nearby characters to your client
  • "shutdown" shuts down the server
  • "startlog" begins logging packet data send to and from your client.
  • "stoplog" stops packet logging. log file in /pol/log/[account name].log
  • "threads" prints thread info
  • "set [priv] [on/off]" turns the privilage on or off

If you know if any information is incorrect on these pages, mail your corrections to polteam@polserver.com
